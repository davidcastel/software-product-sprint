A course project that I worked on while completing my introduction to python3 course. This project was built using Python3 and using what I learned from my course, I was able to build a command-line text-based game in which the user types in one of the following commands ("W", "S", "A", "D") to move the character around the map. The walls are not visible, however, the user can type one of the following commands as promoted on the screen to show where the walls are located and see how to get to the finish line.
With the help of math, I was able to implement a map that tracks the user position without moving the position of the walls.
